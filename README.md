A simple web application built using **Flask** and **SQLite** to manage student records and academic performance.

This project was developed during my **first/second year of university** as part of my journey in learning backend web development with Python and Flask.

The application allows users to add student records, store marks for multiple subjects, view all records, and remove students from the database through a simple web interface.

> **Note:** This is an early learning project created to understand Flask, SQLite, HTML, CSS, and CRUD operations.

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Rishikeshsanin/Student-Performance-Manager.git
```

### Navigate into the project

```bash
cd Student-Performance-Manager
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open in your browser

```
http://127.0.0.1:5000
```

## 📸 Project Overview

The application provides a simple interface where users can:

- Add a student's basic details
- Enter marks for five subjects
- Store records in a SQLite database
- View all student records
- Delete existing records

The database is created automatically when the application is executed for the first time.
